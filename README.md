## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+
- Bun package manager
- SQLite (for local development)

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd dimpy-builds-awesome

# Install frontend dependencies
bun install

# Install backend dependencies
cd backend
bun install
cd ..

# Setup environment variables
cp .env.example .env
```

### Development

```bash
# Start frontend development server
bun run dev

# Start backend development server
bun run backend:dev

# Start both simultaneously
bun run dev:full
```

### Production Build

```bash
# Build frontend
bun run build

# Build backend
bun run backend:build

# Generate sitemap
bun run sitemap
```
